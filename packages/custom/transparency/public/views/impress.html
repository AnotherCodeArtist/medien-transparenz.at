How this web-site came into being

It all started with computer programming project as part of one our courses. This led to a prototypic solution created by the following students:

  • Amir Basyouni
  • Mercedes Stibler
  • Stephan Vrecer

This solution was the basis for the now available site

After winning a netidee-grant for the year 2015, the site developed further. With the help of the following students new features were added:

  • DI Amir Basyouni, BSc
  • DI Gerald Weidinger, BSc

Source of Data

All Data that is published via this site it taken from the quarterly reports published by KommAustria. It was then semi-automatically improved by correcting the most obvious errors using OpenRefine. Although this was meant to significantly improve the data qualtity it could have also introduced some errors.
